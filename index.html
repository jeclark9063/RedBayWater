Welcome to the Franklin County property map

  •  About this site
  •  Technology

The Franklin County Property Map is a development of the Franklin County GIS Consortium, in cooperation with the Franklin County Revenue Department.

Attribution
  • This application was forked from the ieldran project by Katy Meyers, Department of Anthropology, Michigan State University
  • Built from Bootleaf, created by Bryan McBride, a framework for building responsive web mapping applications with Bootstrap 3 and Leaflet
  • Created by Michael Hughes, Earl Clark, and Russ Martelli
  • Data hosted in Carto and ArcGIS Online
Features
  • Fullscreen mobile-friendly map template with responsive navbar, sidebar, and modal placeholders
  • jQuery loading of external GeoJSON files
  • Elegant client-side multi-layer feature search with autocomplete using typeahead.js
  • Map Markers complements of MediaLoot.
  • Custom Location Based URLs complements of Leaflet Hash by Michael Evans.
